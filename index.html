PromptMesh

PromptMesh is a modular ecosystem of open-source tools for building LLM-native infrastructure using the Model Context Protocol (MCP).

EasyMCP

Python SDK for orchestrating MCP servers and tools.

SemanticTool

Semantic discovery and tool invocation microservice with REST endpoints.

RelayMCP

Session-aware OpenAPI gateway for routing REST to MCP tools.
